## How to Join

Want to see your name here? We'd love your help!

### Ways to Contribute

**Code:**
- Fix bugs
- Add features
- Improve performance
- Refactor code

**Documentation:**
- Improve guides
- Fix typos
- Add examples
- Translate

**Community:**
- Answer questions
- Report bugs
- Suggest features
- Share Celstomp

### Getting Started

1. **Fork the repository** on GitHub
2. **Set up development environment**
3. **Find an issue** to work on
4. **Submit a pull request**
