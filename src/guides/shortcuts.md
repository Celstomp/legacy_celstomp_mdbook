# Keyboard Shortcuts

Master these shortcuts to speed up your workflow significantly.

## Essential Shortcuts

These are the shortcuts you'll use constantly:

### Tools

| Key | Tool |
|-----|------|
| `1` | Brush |
| `2` | Eraser |
| `3` | Fill Brush |
| `4` | Fill Eraser |
| `5` | Lasso Select |
| `6` | Rectangle Select |
| `7` | Eyedropper |

### Navigation

| Key | Action |
|-----|--------|
| `Space` | Pan (hold) |
| `+` / `-` | Zoom in/out |
| `Ctrl + 0` | Reset zoom |
| `Home` | Go to first frame |
| `End` | Go to last frame |
| `?` | Show shortcuts |

### Timeline

| Key | Action |
|-----|--------|
| `Space` | Play/Stop |
| `←` / `→` | Previous/Next frame |
| `Q` / `E` | Previous/Next cel |
| `Shift + ←` | Previous 5 frames |
| `Shift + →` | Next 5 frames |
| `L` | Toggle loop |
| `P` | Toggle ping-pong |

### Editing

| Key | Action |
|-----|--------|
| `Ctrl + Z` | Undo |
| `Ctrl + Y` | Redo |
| `Ctrl + X` | Cut |
| `Ctrl + C` | Copy |
| `Ctrl + V` | Paste |
| `Ctrl + D` | Duplicate |
| `Delete` | Clear selection/frame |

### View

| Key | Action |
|-----|--------|
| `O` | Toggle onion skin |
| `F` | Toggle fill mode |
| `H` | Flip horizontal (view) |
| `Tab` | Toggle UI panels |

### Brush Control

| Key | Action |
|-----|--------|
| `[` | Decrease brush size |
| `]` | Increase brush size |
| `Shift + [` | Decrease size (fast) |
| `Shift + ]` | Increase size (fast) |
| `Alt` (hold) | Eyedropper |
| `Shift` (hold) | Straight line constraint |

## Modifier Keys

### Shift Key

Use Shift for:

- **Add to selection** - Shift+select
- **Constrain proportions** - Shift+resize
- **Straight lines** - Shift+draw
- **Nudge 10px** - Shift+arrow keys
- **Frame jump 5** - Shift+timeline nav

### Alt Key

Use Alt for:

- **Eyedropper** - Alt+click (any tool)
- **Subtract selection** - Alt+select
- **Center from point** - Alt+drag
- **Solo layer** - Alt+click layer

### Ctrl Key

Use Ctrl for:

- **Standard shortcuts** - Ctrl+C, Ctrl+V, etc.
- **Precise movements** - Ctrl+drag
- **Select all** - Ctrl+A
- **Deselect** - Ctrl+D

## Custom Shortcuts

### Creating Custom Shortcuts

Celstomp allows some customization:

1. **Edit > Keyboard Shortcuts**
2. **Find action** in list
3. **Click shortcut field**
4. **Press desired keys**
5. **Save configuration**

### Reset to Default

Restore original shortcuts:

1. **Edit > Keyboard Shortcuts**
2. **Click "Reset to Default"**
3. **Confirm**

## Learning Shortcuts

### Start With Essentials

Focus on these first:

1. `Space` - Play/stop
2. `Ctrl + Z` - Undo
3. `1`, `2` - Brush, eraser
4. `[`, `]` - Brush size
5. `O` - Onion skin
6. `?` - Show shortcuts

### Practice Routine

Spend 5 minutes practicing:

1. Switch between tools (1-7)
2. Adjust brush size ([ ])
3. Play animation (Space)
4. Toggle onion skin (O)
5. Use eyedropper (Alt)

### Muscle Memory

Shortcuts become automatic with use:

- Use them consciously for first week
- Force yourself to not click UI
- Soon you'll use them unconsciously
- Speed increases dramatically

## Pro Tips

### Contextual Shortcuts

Some shortcuts change based on context:

- **Space** - Hand tool (normal), Play (timeline focus)
- **Enter** - Confirm (dialogs), Commit (transform)
- **Esc** - Cancel (most operations)

### Hidden Shortcuts

Less obvious but useful:

- **Middle-click drag** - Pan (like Space)
- **Scroll wheel** - Zoom
- **Shift+scroll** - Timeline zoom
- **Double-click layer** - Rename (if enabled)

### Shortcut Combinations

Chain shortcuts for efficiency:

1. `5` (lasso) → select → `1` (brush) → draw
2. `Alt+click` → color → `]` → size → draw
3. `Q` → prev cel → `O` → onion → draw

## Troubleshooting

**Shortcut not working:**
- Check if focus is in text field
- Verify no modal dialog open
- Try clicking canvas first
- Check for conflicts

**Forgot shortcut:**
- Press `?` for quick reference
- Check menu items (show shortcuts)
- Look for tooltip hints
- Reference this guide

**Conflicting shortcuts:**
- Browser may use same shortcuts
- Try with/without Fn key
- Check system shortcuts
- Customize if needed
