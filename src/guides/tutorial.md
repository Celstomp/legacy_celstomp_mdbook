# Built-in Tutorial

Celstomp includes an interactive tutorial that walks you through the basics. This guide explains how to access and use it.

## Accessing the Tutorial

### First-Time Launch

When you first open Celstomp, the tutorial automatically starts. It guides you through:

- The interface layout
- Basic drawing
- Timeline navigation
- Essential shortcuts

### Manual Access

Restart the tutorial anytime:

1. **Help menu** > "Start Tutorial"
2. Or press `?` twice
3. Tutorial resets and begins

### Tutorial Button

Look for the **?** icon in the header to access tutorial options.

## Common Tutorial Questions

**"Can I skip the tutorial?"**
Yes, click "Skip" or press `Esc`. Access later via Help menu.

**"Tutorial is too fast/slow"**
- Go at your own pace
- Each step waits for you
- Take breaks between sections

**"I made a mistake"**
Tutorial is forgiving:
- Try again
- Use Ctrl+Z to undo
- Restart section if needed

**"Tutorial won't start"**
- Check browser compatibility
- Try manual start (Help menu)
- Refresh page
- Check for JavaScript errors

**"Can I do tutorial multiple times?"**
Yes, restart as often as you like. Good for practice.
